i am from, "if you hear about it, you're invited."

grammy clasps my hand in hers,

fingers like tree roots,

every wrinkle an arroyo.

como corazón para casa, like heart for home,

como yo para tú, like me for you,

como pulpa para pan, like flesh for bread,

como comida para amor, like food for love,

ten piedad. have mercy.
